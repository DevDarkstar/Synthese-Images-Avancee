TP Shader Toon & Gooch

Commandes :
't' : affichage du tore avec le shader Toon
'g' : affichage du tore avec le shader de Gooch
'd' : afficher/masquer la silhouette sur le tore
'+' : augmente l'effet de bord de la silhouette
'-' : diminue l'effet de bord de la silhouette
'q' : quitter le programme (pensez à quitter le programme de cette manière afin d'éviter d'éventuelles fuites de mémoire).
